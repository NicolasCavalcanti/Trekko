# Trekko Site (Next.js)

- Node 18/20
- `npm install`
- Local build: `npm run build` (ou `npm run export` para estático)
- Deploy recomendado: **Vercel**. Para GitHub Pages, use `npm run export` e publique o diretório `/out` no branch `gh-pages`.
